Infrastructure is a CLI tool for bundling, compressing, and managing files — with its own open file formats and a visual GUI called Infrastructure Wizard.

bash — infrastructure
# Pack a folder into an IDF bundle
$ node infra.js pack ./myproject -o bundle.idf
✓ Packed 14 files into bundle.idf
 
# Compress to zip
$ node infra.js compress ./dist -o archive.zip
✓ Done! 48320 bytes written to archive.zip
 
# Launch the visual Wizard
$ npm run wizard
✓ Infrastructure Wizard is running

File Formats

Infrastructure's own open formats

Infrastructure introduces two custom file formats, designed from scratch for clarity and portability.

.idf

Infrastructure Decompressable Folder

A plain-text bundle format that stores multiple files and their full directory structure inside a single file. Includes a human-readable header, a file table listing every entry with its size, and the raw file contents — all in one place. Designed to hold large collections of files.

.ipf

Infrastructure Project File

A compressed multi-file JavaScript project format. Write your code in Infrastructure Wizard, save it as a tiny .ipf — the file contents are gzip-compressed, making it significantly smaller than storing the raw files. Load it back into the Wizard at any time to continue editing.

Infrastructure Wizard

A full desktop GUI for Infrastructure, built with Electron. No terminal required — just open the Wizard and use everything through a clean visual interface.

  • Pack, unpack, compress and bundle via forms
  • Built-in multi-file code editor with tabs
  • Save and load .ipf project files
  • Run JavaScript directly from the editor
  • Live output log for all operations
  • Dark terminal-style aesthetic

Dependencies

What Infrastructure is built on

Infrastructure uses a small set of well-maintained open source packages. Here's exactly what's under the hood.

commander
^13.1.0
CLI argument and command parsing. Powers all of Infrastructure's terminal commands and flags.
archiver
^6.0.2
Streaming archive creation. Handles zip compression with maximum zlib compression level.
fs-extra
^11.3.5
Enhanced file system operations. Used for reading, writing, and ensuring directory paths exist.
chalk
^4.1.2
Terminal string styling. Gives Infrastructure's output coloured success, warning, and info messages.
glob
^11.0.2
File pattern matching. Used to walk directory trees when packing folders into IDF files.
electron
^36.3.1
Desktop app framework. Powers Infrastructure Wizard — the visual GUI for the tool. (Dev dependency.)
Get started

Install Infrastructure

Requires Node.js. Clone the repo, install dependencies, and you're ready to go.

1. Clone the repository
git clone https://github.com/your-username/infrastructure
2. Install dependencies
npm install
3. Run a command
node infra.js --help
4. Launch the Wizard
npm run wizard
